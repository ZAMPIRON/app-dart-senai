# 📱 SENAI Mobile - APP em DART

https://github.com/user-attachments/assets/548f2130-a367-4d9d-93c0-542198dec1a3

Este repositório contém o código-fonte de um sistema de autenticação mobile desenvolvido para o ecossistema **SENAI**. O projeto foca em uma interface intuitiva, transições suaves e uma arquitetura limpa utilizando a linguagem **Dart**.

## 🚀 Tecnologias Utilizadas

* **Dart:** Linguagem de programação otimizada para o desenvolvimento de interfaces rápidas em qualquer plataforma.
* **Flutter:** Framework UI para criação de aplicativos compilados nativamente. (Caso tenha usado)
* **Material Design:** Princípios de design seguidos para garantir uma experiência de usuário consistente.

---

## 🛠️ Funcionalidades do Protótipo

O vídeo demonstra o fluxo completo de gerenciamento de acesso:

### 1. Tela de Login

* Autenticação via E-mail e Senha.
* Navegação rápida para as telas de criação de conta e suporte.

### 2. Tela de Cadastro

* Coleta de dados estruturada: Nome, Sobrenome, E-mail e Telefone.
* Sistema de criação de senha com confirmação para evitar erros de digitação.
* Link direto para retornar ao login caso o usuário já possua conta.

### 3. Recuperação de Senha

* Fluxo simplificado para reinicialização de credenciais via e-mail.

## 🎨 Identidade Visual

O projeto respeita rigorosamente o manual de marca do **SENAI**, utilizando:

* **Cores Primárias:** Vermelho Institucional e Azul Marinho.
* **Layout:** *Mobile First*, com foco em acessibilidade e facilidade de clique nos inputs.

## 📂 Como Executar o Projeto

1. Certifique-se de ter o **Flutter SDK** e o **Dart** instalados em sua máquina.
2. Clone este repositório:
```bash
git clone https://github.com/seu-usuario/seu-repositorio.git

```


3. Navegue até a pasta do projeto e instale as dependências:
```bash
flutter pub get

```


4. Execute o aplicativo:
```bash
flutter run

```
